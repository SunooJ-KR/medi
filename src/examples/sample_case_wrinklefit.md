# 실제 사례 검증 — 링클핏(WrinkleFit) 라인 / 일본(JP)

> 작업 4-2 산출물. 실제 한국어 제품 설명 입력 → 위반 탐지 → 수정 → PASS 전환 과정 기록.
> 룰셋: `rules/jp.json` v1.1.0 (JP-005 若返り, JP-006 シワ除去·改善 추가).

## 입력 (원본 한국어 카피)

```
주름을 완벽하게 지워주는 링클핏 크림. 피부 재생 성분이 세포부터 되살려 10년 젊어진 피부로. 일본 판매 1위 안티에이징 크림입니다.
```

## STAGE 2 — 위반 탐지 (엔진 실제 출력)

```
python src/scripts/validate_copy.py --input "{원본}" --rules src/rules/jp.json --today 2026-07-07
```

```
verdict   VIOLATION
pass_rate 0/3 문장 통과
JP-006  주름을 완벽하게 지워   VIOLATION   (シワ除去 — 효능 범위 초과)
JP-003  재생                  VIOLATION   (再生 — 치료적 표현)
JP-003  되살                  VIOLATION   (蘇生 — 치료적 표현)
JP-005  10년 젊 / 젊어진       VIOLATION   (若返り — 연령 역행)
JP-005  안티에이징            VIOLATION   (若返り 개념 브리지)
JP-004  1위                   WARNING     (景表法 — 근거 필요)
```

- 한국어 원본이지만 룰의 ng_patterns는 일본어(`再生`/`若返り`) 중심 → **개념 브리지**(`안티에이징`→`若返り`)와 한국어 병기 패턴으로 교차언어 탐지.
- "주름 완벽하게 지워"는 JP-006 정규식 `주름.{0,10}(지워|…)`이 문장 내 삽입어("완벽하게")를 건너뛰고 매칭.

## 수정 (alternatives 기반 재작성 → JP 준수 카피)

```
乾燥による小じわを目立たなくするリンクルフィットクリーム。うるおいでキメを整え、ハリのある肌印象へ。年齢に応じたうるおいケアを。
```

## STAGE 2 재검증 — PASS 전환 (엔진 실제 출력)

```
verdict   PASS
pass_rate 3/3 문장 통과
```

## 전환 요약

| 단계 | verdict | 통과율 |
|------|---------|--------|
| 원본 | 🔴 VIOLATION | 0/3 |
| 수정 | 🟢 PASS | 3/3 |

- 치료적/연령역행/효능초과 표현 3종(JP-003/005/006)을 화장품 허용 범위("乾燥による小じわを目立たなくする" 등)로 교체해 전건 해소.
- 룰셋 확장(JP-005/006) 후 기존 단위 테스트 14건 무회귀 확인.
