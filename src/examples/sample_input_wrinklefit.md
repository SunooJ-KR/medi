# 예제 입력 — 링클핏(WrinkleFit) 크림

> LaunchPass 스킬 입력 예제. **국가는 실행 시 지정**한다(데이터 사전 내장 없음).
> SKILL.md `## 입력` 계약: `product_name`·`product_desc`·`target_country` 필수, `ko_copy` 선택.

## 입력 필드

- **product_name**: 링클핏 리페어 크림 (WrinkleFit Repair Cream)
- **target_country**: (실행 시 지정 — 예: `일본`, `미국`, `베트남` …)
- **product_desc**:
  - 카테고리: 안티에이징 페이스 크림
  - 핵심 성분: 레티놀 유도체, 펩타이드, 히알루론산
  - 소구 효능: 주름·탄력·건조 케어
  - 타겟: 30~50대
- **ko_copy** (검증할 한국어 마케팅 카피):
  ```
  주름을 완벽하게 지워주는 링클핏 크림. 피부 재생 성분이 세포부터 되살려 10년 젊어진 피부로. 일본 판매 1위 안티에이징 크림입니다.
  ```

## 실행 예시 (자연어 트리거)

```
링클핏 리페어 크림 일본 런칭 검증해줘.
카피: "주름을 완벽하게 지워주는 링클핏 크림. 피부 재생 성분이 세포부터 되살려 10년 젊어진 피부로. 일본 판매 1위 안티에이징 크림입니다."
```

- `target_country`를 `미국`/`베트남` 등으로 바꾸면, 해당 국가 데이터가 없을 경우 **부트스트랩**(concerns→rules→personas draft 생성→승인)이 먼저 수행된 뒤 동일 파이프라인이 실행된다.
- 실제 검증 결과: `sample_case_wrinklefit.md`(일본) 참조.
